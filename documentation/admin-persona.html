Persona: Jordan Davis

Role

Individual working as an Admin for this organization who is looking for a way to compile the database in a more convenient matter, so it is viewable.

Technology

Cellphone:

  • iPhone 12 iOS 15.4.1

Computer:

  • MacBook Air iOS 11

Proficiency:

  • Admin user when it comes to operating his devices. Is in charge of monitoring the database with his devices. Is a quick learner and stays up to date with updates in the tech world.

Love/Hate

  • Loves getting fast information
  • Loves having information easily displayed
  • Hates not having a way to view database easily

Why Use the Site?

  • A quick and easy way to view the compiled database as well as see it added to
  • Will make the information accessible to more average users

Frustrations

  • Not having a proper way to display the database of information that has built up
  • Not being able to use this data to get information across to others

SignUp Story:

As an admin user, I need a way to display and store our database of information and photos in an easy-to-use matter where I can than begin to manage what information we wish to display for the public.

Use Case

Jordan is looking to display his organizations information and pictures in a user-friendly manner. He wants to make this information available to the public, who can not only view this information but also add their own images with information in order to help with the goal of archiving how landscapes around New Mexico have changed over time to get valuable data.

Precondition:

Jordan is logged into the website on the home page.

Post Condition:

Jordan successfully uploads data and pictures from his database.

Interaction Flow>

  • Jordan goes to the tab that says "Upload"
  • Website loads page to directly upload data and images
  • Jordan links a picture from his data base
  • Website pops up a display window showing the image
  • Jordan begins filling out information related to the image such as location and date
  • Website begins compiling information to store with image
  • Jordan hits "Submit"
  • Website loads the main page showing the addition of the image and information to the map via a pin.
